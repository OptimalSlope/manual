## Simulation Workflow

Follow these steps to configure properties for a selected section in OptimalSlope and run a simulation.

This workflow is split into short tutorials. Each page explains one part of the process and includes a short video demonstration.

### Workflow steps

1. [Importing or Generating Sections]({{ site.baseurl }}/pages/Tutorials/Workflow/1-importing-or-generating-sections/)
2. [Define Section Properties]({{ site.baseurl }}/pages/Tutorials/Workflow/2-define-section-properties/)
3. [Running Simulations]({{ site.baseurl }}/pages/Tutorials/Workflow/3-running-simulations/)
4. [Exporting Output Profile to DXF]({{ site.baseurl }}/pages/Tutorials/Workflow/4-exporting-output-profile-to-dxf/)
5. [Importing Output into Rockscience RS2]({{ site.baseurl }}/pages/Tutorials/Workflow/5-importing-output-into-rs2/)
